# 27 เมษายน 2569

รายงานนี้รวบรวมการเปลี่ยนแปลงทั้งหมดที่เกิดขึ้นในส่วนของหน้าหลักสูตร (Courses) และหน้าสาขาความเชี่ยวชาญ (Expertise) เพื่อพัฒนาให้ระบบมีความสมบูรณ์ สวยงาม และใช้งานได้ง่ายขึ้น

---

## 1. ฟีเจอร์ใหม่: หน้าหลักสูตรทั้งหมด (All Courses)
*   **เส้นทางใหม่:** พัฒนาหน้าแสดงรายการหลักสูตรทั้งหมดที่ `/consumer-protection/all-courses`
*   **ดีไซน์พรีเมียม (Hero Section):** 
    *   ใช้โครงสร้างแบบเดียวกับหน้ารายละเอียดหลักสูตร (Hero Image + Overlay)
    *   เพิ่ม Badge หมวดหมู่ "สถาบันการศึกษา" และหัวข้อขนาดใหญ่ที่อ่านง่าย
*   **ระบบค้นหาและกรองข้อมูล (Search & Filter):**
    *   **Search:** ระบบค้นหาชื่อหลักสูตรแบบ Real-time
    *   **Filters:** แท็บสำหรับกรองหลักสูตรตามระยะเวลา (สั้น, กลาง, ยาว) หรือดูทั้งหมด
*   **การเชื่อมโยงข้อมูล:** ลิงก์ "ดูทั้งหมด" จากหน้าหลักจะส่งค่า Query Parameter (`?cat=...`) เพื่อเลือกแท็บที่ถูกต้องให้โดยอัตโนมัติเมื่อเปลี่ยนหน้า

## 2. การปรับปรุงระบบนำทางและสถานะ (Navigation & Logic)
*   **Smart Back Button (router.back()):**
    *   เปลี่ยนปุ่มย้อนกลับในหน้ารายละเอียดและหน้า All Courses จากลิงก์ตายตัว เป็นการใช้ History API
    *   **ประโยชน์:** จดจำได้ว่าผู้ใช้มาจากหน้าไหนและพากลับไปที่หน้านั้นอย่างถูกต้อง (เช่น จาก All Courses -> Detail -> กลับไป All Courses)
*   **Tab State Persistence (Tab Sync):**
    *   อัปเดต `SidebarTabsSection` ให้ซิงค์สถานะแท็บปัจจุบันกับ URL ผ่าน `?tab=...`
    *   **ประโยชน์:** เมื่อผู้ใช้กด "ย้อนกลับ" จากหน้ารายละเอียด ระบบจะเปิดแท็บเดิมที่เคยกดค้างไว้ให้ทันที ไม่กลับไปแท็บเริ่มต้น
*   **ปรับปรุงคำบรรยาย:** เปลี่ยนข้อความปุ่มจาก "กลับหน้าหลัก" เป็น **"ย้อนกลับ"** เพื่อความเข้าใจที่เป็นสากล

## 3. การยกระดับ UI/UX และความสวยงาม (Design Polish)
*   **ระบบ Tooltips อัจฉริยะ:**
    *   เพิ่มข้อความ "คลิกเพื่อดูรายละเอียด" เมื่อ Hover บนการ์ดหลักสูตร
    *   **การจัดวาง:** วางไว้ที่มุมขวาบนนอกการ์ด (Floating Above) เพื่อไม่ให้บังข้อความสำคัญ
    *   **Animation:** เพิ่มการเคลื่อนไหวแบบยกตัว (Lift Animation) เมื่อเอาเมาส์ชี้
*   **การจัดการขอบมน (Border Radius Fix):**
    *   แก้ไขปัญหาขอบเหลี่ยมที่เกิดขึ้นเมื่อ Hover บนการ์ด (เนื่องจากการใช้ `overflow: visible`)
    *   **วิธีแก้:** กำหนดความโค้งมนที่ส่วนหัว (Header) และส่วนท้าย (Footer) ของการ์ดให้สัมพันธ์กับตัวการ์ดหลัก
*   **Icons & Assets:**
    *   อัปเกรดไอคอนทั้งหมดเป็น `react-icons/fa6`
    *   แก้ไขรูปภาพสาขาความเชี่ยวชาญที่แสดงผลผิดพลาด (`card[id].jpg`)
*   **ระบบ Hover Overlay ในหน้าสาขา (Expertise):**
    *   เปลี่ยนจาก Tooltip ลอย เป็นระบบ **Hover Overlay** (เบลอภาพพื้นหลังและแสดงปุ่ม "ดูรายละเอียด" ตรงกลาง)
    *   ใช้เทคนิค `backdrop-filter: blur` เพื่อความพรีเมียม และเพิ่มการซูมภาพเล็กน้อย (Scale) เมื่อ Hover
